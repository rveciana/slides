Python per modelització

Python: Què és

  • Un llenguatge interpretat
  • Tipus dinàmics / Tipus forts
  • Orientat a objectes
  • Multi-propòsit
  • Pensat per a ser ràpid (d'aprendre, de fer servir, d'entendre)
  • Pensat per a crear un codi net i uniforme

Python: Alguns avantatges

  • Permet resoldre problemes simples de manera senzilla, però també problemes complexes
    • Codi tipus script
    • Programació orientada a objectes amb excepcions, etc
    • Programació funcional
  • Llibreria standard molt gran: "piles incloses"
  • Moltes llibreries i frameworks que l'extenen
    • Creació de pàgines web
    • Administració de sistemes
    • Llibreries geogràfiques i científiques
    • Programació en paral·lel
    • Optimització del codi amb Cython
  • Python enganxa llibreries programades en diferents llenguatges
  • Software que permet fer-lo servir interactivament: ipython/jupyter, SAGE

Funcions i utilitats incloses

  • logs amb logging
  • ssh i sftp amb paramiko (sense intercanvi de claus)
  • Connexions amb les bases de dades
  • Baixada de pàgines web/recursos API
  • Lectura i escriptura d'arxius JSON i XML
  • ...

Algunes llibreries de python útils per a modelització

  • NumPy per càlcul amb matrius
  • Scipy extén NumPy amb functions com les de MATLAB, IDL, Octave, R-Lab i SciLab
  • Matplotlib dibuix de gràfiques
  • Basemap extén Matlpotlib per a dibuixar mapes
  • GDAL/OGR Geospatial Data Abstraction Library
    • Fiona
  • PyGRIB per a llegit fitxers GRIB
  • Pandas per lectura de dades
  • Scikit-learn: machine learning

GDAL/OGR

Geospatial Data Abstraction Library
  • Model de dades abstracte per llegir formats GIS ràster i vectorials
  • Més de 150 formats suportats, entre ells:
    • GeoTIFF
    • ENVI
    • Shapefile
    • PostGIS
    • GRIB
    • NetCDF
    • IRIS
  • Fet servir per moltes aplicacions, està escrit en C++, i té bindings en Python
  • Llegeix/Escriu/Canvis de coordenades/Operacions GIS

Format VAISALA IRIS

  • Format propietari de SIGMET/VAISALA per radars meteorològics
  • Dificultat per accedir a les dades
  • Driver GDAL fet al'SMC

Eines i tècniques recomanables

  • PyCharm

    • Control de versions svn / git
    • Test Driven Development (TDD)
